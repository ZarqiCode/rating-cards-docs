04 — TYPOGRAPHY

### Two voices, one system

Cormorant Garamond speaks at display scale — emotive, editorial, unmistakable. Jost listens at body

scale — quiet, geometric, precise. One speaks, one supports. They never compete.

CORMORANT GARAMOND — Display typeface · Google Fonts · Free

```
88px / Light
```
#### Hero headline

```
56px / Light Italic
```
##### Taglines & pull quotes

32px / Regular Section headlines

22px / Light Card headlines

JOST — Body & UI typeface · Google Fonts · Free

300 / Light Body copy, descriptions, longer text passages

400 / Regular UI labels, button text, navigation

500 / Medium KICKERS · LABELS · OVERLINES

TYPOGRAPHY RULES

```
Scale threshold
Cormorant Garamond is
display-only. Never use it below
20px — it becomes illegible. Jost
handles everything below that
threshold.
```
```
Italic as accent
The italic of Cormorant is used
for emotional words and taglines
only — never for full paragraphs.
One italic moment per
composition maximum.
```
```
Letter spacing
Jost uppercase labels always
use 0.12–0.20em tracking.
Cormorant display sizes use
default tracking or very slight
negative at large sizes.
```
05 — VOICE & TONE

### How we speak

Our voice is the personality behind the product. It is warm but never sycophantic. Confident but never

arrogant. Precise but never clinical. We speak like someone who genuinely understands café culture.


We are

- Calm and assured — we don't shout
- Human and personal — always "you", never "the
    user"
- Precise — short sentences, no filler words
- Warm — we understand the 5am espresso pull
- Knowledgeable — we speak hospitality fluently

We are not

- Excitable — no "Amazing!" or "Wow, great news!"
- Corporate — no "leverage synergies" or "unlock
    value"
- Pushy — we never create false urgency
- Sycophantic — no excessive congratulations
- Technical — we explain outcomes, not features

PLAIN ENGLISH — THE SHOPKEEPER RULE

Our voice is calm and warm, but it must also be **immediately understood by a busy
café owner skimming on their phone between shifts**. Marketing copy that sounds
fancy at the cost of clarity is failing the brief. The shopkeeper rule keeps the
warmth without the fog.

```
LEAD WITH WHAT IT IS
First sentence: name the thing. Second
sentence: what it does. Third sentence: the
outcome for the owner. In that order. Always.
```
```
DROP UNEARNED ADJECTIVES
"Premium" stays — it's our positioning.
"Smart", "seamless", "powerful", "innovative",
"revolutionary" go — they are filler that hides
the actual product.
```
```
ONE ITALIC PER SECTION
Display copy gets exactly one italic moment,
on the emotional beat. More than one and the
emphasis stops working.
```
```
SHORT SENTENCES
Aim for ~12 words. If a sentence runs longer,
split it. Two short sentences always beat one
long one in marketing copy.
```
